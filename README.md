# GameEngine

## 프로젝트 개요

이 저장소는 2026 Spring `GameProgramming` 강의의 simple game engine 과제를 위한 작업 공간입니다.
현재 목표는 게임 엔진의 핵심 구성 요소를 단계적으로 구현하며, 각 단계에서 어떤 개념을 적용했는지 명확히 기록하는 것입니다.

## 현재 진행 상태

현재는 `game loop` 동작 원리를 적용해 간단한 육망성을 조작하는 단계까지 진행되었습니다.
입력, 갱신, 렌더링의 흐름을 분리해서 다루는 기본 구조를 확인하는 것이 이번 단계의 핵심입니다.

## 진행 과정

### 1. 프로젝트 시작

- Visual Studio 기반 C++ 프로젝트를 구성했습니다.
- 강의 과제용 simple game engine 저장소를 정리했습니다.

### 2. 현재 단계

- game loop의 반복 구조를 프로젝트에 반영했습니다.
- 간단한 육망성을 화면에서 조작하는 단계까지 구현을 진행했습니다.
- 입력 처리와 프레임 단위 갱신 흐름을 이해하고 확인하는 데 초점을 두고 있습니다.

### 3. 다음 업데이트에 기록할 내용

- 현재 단계에서 무엇을 구현했는지
- 어떤 엔진 개념을 새로 적용했는지
- 이전 단계와 비교해 무엇이 달라졌는지
- 다음 단계에서 다룰 목표가 무엇인지
